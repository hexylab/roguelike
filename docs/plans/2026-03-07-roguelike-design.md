# ローグライクゲーム設計書

## 概要
Pyxelで作る不思議のダンジョン風ローグライクゲーム。

## コア仕様
- 画面: 320x240（マップ240x240 + 右パネル80x240）
- タイル: 16x16
- マップ表示: 15x15タイル（視界中心スクロール）
- ダンジョン: BSP法、10F
- 移動: 8方向ターン制
- アセット: DawnLike 16x16 tileset (CC-BY-SA 3.0)

## アイテムシステム
- 武器・盾（+強化値）
- 回復薬・巻物（未識別システム）
- 投擲可能
- インベントリ最大20枠

## 戦闘
- ダメージ = 攻撃力 - 防御力/2（最低1）
- フロアごとに敵が強化

## 敵テーブル
